## Use it local

### 1. Instal node.js on your computer
You need install node.js: https://nodejs.org/en/

### 2. Clone the project
You need to clone the https://github.com/DominikWysota/notesToDO.git

### 3. Open project
```
cd ./notesToDo
npm i
cd ./client
npm i
cd ..
```
### 4. Start project in localhost
```
npm run dev
```
